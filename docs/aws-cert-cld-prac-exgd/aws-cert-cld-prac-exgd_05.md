+   *第七章*，*AWS 计算服务*

+   *第八章*，*AWS 数据库服务*

+   *第九章*，*AWS 上的高可用性与弹性*

+   *第十章*，*应用集成服务*

+   *第十一章*，*AWS 上的分析*
