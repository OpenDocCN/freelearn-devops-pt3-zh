+   *第六章*，*使用 GitHub Actions 实现自动化*

+   *第七章*，*运行你的工作流*

+   *第八章*，*使用 GitHub Packages 管理依赖关系*

+   *第九章*，*部署到任何平台*

+   *第十章*，*功能开关与功能生命周期*

+   *第十一章*，*基于主干的开发*
